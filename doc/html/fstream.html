Purely functional parsers

Purely functional parsers are an alternative of parsers of streams where the stream type used is a lazy non-destructive type. These streams are lazy values, like in classical parsers of streams, but the values are not removed as long as the parsing advances. To make them work, the parsers of purely functional streams return, not the simple values, but a couple of the value and the remaining stream.
